Type Selector: p{}

Universal Selector: *{}
Attribute Selector

Presence and value attribute selectors

[attr]{}: have the attribute attr, whatever it's value

data-test
[data-test]{color: black;}

[attr=val]{}: only attr === val

data-test='hello'
[data-test='hello']{color: red;} //match
data-test='hello hi'
[data-test='hello']{color: red;} //mismatch

[attr~=val]{}: only the attribute val is one of the 'space-separated' list

data-test='world'
[data-test~='world']{color: blue;} //match
data-test='world test'
[data-test~='world']{color: blue;} //match
data-test='world!!!'
[data-test~='world']{color: blue;} //mismatch

Substring value attribute selectors

[attr^=val]{}: attr must start with val

data-test='yourbro'
[data-test^='your']{color: yellow;} //match
data-test='your bro'
[data-test^='your']{color: yellow;} //match
data-test='you r bro'
[data-test^='your']{color: yellow;} //mismatch

[attr$=val]{}: attr must end with val

data-test='he he da'
[data-test$='da']{color: orange;} //match
data-test='heheda'
[data-test$='da']{color: orange;} //match
data-test='hehed a'
[data-test$='da']{color: orange;} //mismatch

[attr*=val]{}: value contains val

data-test='my name is tom'
[data-test*='da']{color: orange;} //match
data-test='mynameistom'
[data-test*='da']{color: orange;} //match
data-test='my name i s tom'
[data-test*='da']{color: orange;} //mismatch
