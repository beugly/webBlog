display: flex

display: flex is a one-dimensional layout method for laying out items(direct children) in rows or columns.

properties for flex container:

flex-direction: specifies the direction. value is column or row
  • row: the default value. the main axis flex items are laid out in horizontal
  • row-reverse: like row but main aixs flex items are permuted
  • column: the main axis flex items are laid out in vertical.
  • column-reverse: like column but main aixs flex items are permuted
flex-wrap:
  • nowarp: the default value. the flex items are laid out in single line.
  • warp: the flex items break into multiple lines.
  • warp-reverse: like warp, but cross-flex-items are laid out from right to left or from bottom to top

shorthand:

flex-flow: flex-direction flex-wrap; for example: flex-flow: row warp;

properties for flex items:

flex: flex-grow flex-shrink flex-basis
  • flex-grow: 1 the flex grow factor of flex item.
  • flex-shrink: 0.5 the flex shrink factor of flex item.
  • flex-basis: 200px the initial size of flex item that are laid out in main aixs.

Horizontal and Vertical alignment
align-items: effect the flex items sit on the cross axis. but set the align-self: flex-end; on flex item can override this.
  • stretch: the default, stretches all flex items to fill the parent in the direction of the cross axis.
  • center: stretches all flex items to fill the parent in the direction of the cross axis.
  • ... (guides)
justify-content: effect the flex items sit on the main axis.
  • flex-start: the default. makes all flex items sit at the flex-start of the main axis.
  • flex-end: makes all flex items sit at the flex-end of the main axis.
  • center: makes all flex items sit at the center of the main axis.
