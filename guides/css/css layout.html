css layout recap

display

display property allow us to change the default way something displays

  • block: block level, width = 100% of it's parent element, can set size
  • inline: inline level, width = content width, height = content height, can't set size
  • inline-block: inline-block level
  • flex: apply flex to layout child elements in one dimesion--either row or column.(display: flex used to parent container)
  • grid: apply flex to layout child elements in two dimesions--lining child elements up in rows and columns.(display: grid used to parent container)
float

set float make the element is moved to the left or right and removed from normal flow, and the surrounding content floats around the floated element.

position

  • static: default value. not position in any special way;
  • relative: behaves the same as static until you add some extra properties;
  • absolute: relative to the nearst positioned ancestor; out of the normal flow
  • fixed: relative to the viewport; out of the normal flow;
  • sticky: behaves the same as static. until you add some extra properties it like fixed

FlexBox

display: flex set on parent element.
flex: 1 0.5 200px; set on flex items(children element).
flex: flex-grow flex-shrink flex-basis is the shorthand
flex-grow: 1 the flex grow factor of flex item.
flex-shrink: 0.5 the flex grow factor of flex item.
flex-basis: 200px the initial main size of flex item.
