Float vs Position: absolute
  • float element remain a part of the normal flow. absolute element are removed from the normal flow.
clear floats:
  • Add Empty Div with the css code style="clear:both;" after the float element.
  • Overflow , setting the overflow to hidden or auto on its parent element
  • :after{display:block;clear:both}
Floats
Properties for flex container:
  • display: flex | inline-flex
  • flex-direction: row | row-reverse | column | column-reverse
  • flex-wrap: nowrap | wrap | wrap-reverse
  • flex-flow: flex-direction flex-wrap the shorthand.
  • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly align all the flex items along the main axis.
  • align-items: flex-start | flex-end | center | stretch | baseline align all the flex item along the cross axis.
Properties for flex items:
  • order: number controls the order of flex item appear in the flex contianer.
  • flex-grow: number grow factor. default 0.
  • flex-shrink: number shrink factor when overflow. default 1.
  • flex-basis: length init size.
  • flex: none | flex-grow flex-shrink flex-basis the shorthand.
  • align-self: flex-start | flex-end broken align-items

Note: float clear vertical-align have no effect on a flex item.
