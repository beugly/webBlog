Box Model

box model rectangle & box-sizing

model box rectangle:

margin + border + padding + content

box-sizing:

  • content-box: width = content width
  • border-box: width = border + padding + content width
box-sizing: content-box
width: 480px
height: 360px
box-sizing: border-box
width: 480px
height: 360px
margin border padding content margin border padding content

Note:
giving above two box the same width, padding, border and different box-sizing

overflow:

  • auto: overflow content is hidden, the scroll bars are shown;
  • hidden: overflow content is hidden without scroll bars
  • visible: overflow content is shown out side the box without scroll bars

background-clip:

  • border-box: the bg extends to the outside edge of the border
  • padding-box: the bg extends to the outside edge of the padding
  • content-box: the bg within the content box
background-clip: border-box
background-clip: padding-box
background-clip: content-box

margin collapsing:

if two adjacent elements both have the margin set on them. and the two margins touch, the larger of the two remains, and the smaller one disappears. (in vertical, not horizontal)

margin-bottom:10px;
margin-top:20px;
the two div, use the larger one: margin-top:20px;
